VIVA QUESTIONS

🔹 Process Management

4. What is a process?
A process is a program in execution.

5. What is a thread?
A thread is a lightweight process that shares memory with other threads.

6. What is scheduling?
It is the method used by OS to decide which process runs next.

7. Types of scheduling algorithms?
FCFS, SJF, Round Robin, Priority Scheduling.

🔹 Memory Management

8. What is RAM?
Temporary memory used by the system to store data for currently running processes.

9. What is virtual memory?
A technique that uses disk space as an extension of RAM.

10. What is paging?
Memory management technique that divides memory into fixed-size pages.

🔹 Deadlock

11. What is deadlock?
A situation where processes are stuck waiting for each other and none can proceed.

12. Conditions of deadlock?
Mutual exclusion, hold and wait, no preemption, circular wait.

🔹 Disk Scheduling

17. What is disk scheduling?
Method to decide order of disk requests.

18. Name disk scheduling algorithms.
FCFS, SSTF, SCAN, C-SCAN.

19. What is seek time?
Time taken to move disk head to required track.

🔹 System Call

Interface between user program and OS.
Used to request OS services.
